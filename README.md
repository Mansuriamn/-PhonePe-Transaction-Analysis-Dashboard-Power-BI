# 📌 Project Overview

India has witnessed rapid growth in digital payments over the last few years. Millions of transactions occur daily across states, districts, and payment categories.

This project analyzes PhonePe transaction and user data to uncover meaningful insights into digital payment trends, customer adoption, regional performance, and transaction growth.

Using Power BI, I designed a **2-page interactive dashboard** that enables users to explore transaction volume, payment categories, regional performance, registered users, app engagement, and yearly growth through dynamic filters and KPI-driven reports.

The objective was not only to visualize the data but also to answer meaningful business questions that could support strategic decision-making.

---

# 🎯 Business Objectives

- Monitor digital payment growth across India.
- Analyze transaction trends over multiple years.
- Identify high-performing states and regions.
- Understand user adoption and app engagement.
- Compare different transaction categories.
- Enable interactive exploration through dynamic filters.
